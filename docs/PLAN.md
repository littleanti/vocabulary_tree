## 🔄 기술 부채 / 개선점

| 항목 | 우선순위 | 메모 |
|---|---|---|
| TypeScript 마이그레이션 | High | state/db 스키마가 빠르게 커짐 |
| Vitest 유닛 테스트 | High | SRL/스냅 거리/스테이지 로직은 테스트 필수 |
| Playwright E2E | Medium | 일일 사이클 + 락 시나리오 |
| Vite 빌드 도구 | Medium | Service Worker 캐시 버스팅 자동화 |
| d3-hierarchy 부분 임포트 | Low | 트리 레이아웃 복잡도 증가 시 |
| 다국어 i18n 구조 | Low | 외국인 학습자 모드 진입 시 |
